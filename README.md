# 📨 Chatterly Backend

**Chatterly** — это собственный мессенджер, вдохновлённый Telegram.  
Данный репозиторий содержит **серверную часть** приложения, реализованную на **NestJS**, с использованием **Prisma ORM**, **PostgreSQL** и **WebSocket** для обмена сообщениями в реальном времени.

---

## 👀 Что это такое?

Это API + WebSocket сервер для мессенджера, который:
- Позволяет пользователям регистрироваться и авторизоваться
- Создавать чаты (личные и групповые)
- Обмениваться сообщениями в реальном времени
- Поддерживает статус "онлайн", индикатор "печатает..." и прочее
- Использует REST API для запросов и WebSocket для real-time событий

---

## 🧰 Технологии / Безопасность

| Технология     | Для чего                                                     |
|----------------|--------------------------------------------------------------|
| **NestJS**     | Основной backend-фреймворк                                   |
| **PostgreSQL** | Реляционная база данных                                      |
| **Prisma ORM** | Работа с БД (миграции, запросы, типобезопасность)            |
| **WebSocket**  | Реальное время для сообщений и событий                       |
| **Docker**     | (опционально) для контейнеризации проекта                    |
| **Argon2**          | Основной backend-фреймворк                                   |
| **Session-based**   | Авторизация пользователей                                    |
| **Class Validator** | Валидация входных данных                                     |
